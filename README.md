# esteh-franchise-system
Sistem manajemen franchise es teh berbasis web yang mencakup pengelolaan cabang, produk, stok, dan transaksi


### Daftar Anggota dan Tanggung Jawab

- **Rafi**  
  Bertanggung jawab atas pengembangan fitur **Login** dan **Dashboard Pusat**.

- **Rafly**  
  Bertanggung jawab atas pengembangan fitur **Produk**, **Dashboard Cabang**, dan **Barang Keluar**.

- **Fariel**  
  Bertanggung jawab atas **Manajemen Barang**.

- **Nabila**  
  Bertanggung jawab atas modul **Transaksi**.

- **Khansa**  
  Bertanggung jawab atas modul **Karyawan**.

- **Ridwan**  
  Bertanggung jawab atas modul **Barang Masuk**.

### Peran Khusus

- **Rafly**  
  Berperan sebagai **Integrator**, yaitu mengelola file universal (seperti konfigurasi database dan komponen bersama), melakukan integrasi seluruh fitur, serta menggabungkan hasil kerja anggota ke branch utama.

### Catatan Pengembangan

Setiap anggota bekerja pada branch masing-masing sesuai dengan tugas yang telah ditentukan.
